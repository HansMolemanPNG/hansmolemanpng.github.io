# hansmolemanpng.github.io

Personal security research blog built with MkDocs + Material theme, hosted on GitHub Pages.

## Setup

1. Clone the repo
2. Install dependencies: `pip install mkdocs-material mkdocs-git-revision-date-localized-plugin`
3. Local preview: `mkdocs serve`
4. Deploy: push to `main` — GitHub Actions builds and deploys automatically

## Repo structure

```
docs/
  index.md              # Blog homepage (post grid auto-generated by hooks.py)
  about.md              # About page
  assets/
    extra.css           # Custom styles
    extra.js            # Filter logic + card click handler
    profile.jpeg        # Profile avatar
  blog/
    <slug>/
      index.md          # Post content
      img/              # (optional) Screenshots and images for this post

overrides/
  main.html             # Custom layout (sidebar + main)

hooks.py                # MkDocs hook: generates post grid and article headers
mkdocs.yml              # MkDocs config
```

## Writing a post

Create a folder under `docs/blog/` with a slug and add `index.md`:

```
docs/blog/my-new-post/index.md
```

Required frontmatter:

```yaml
---
title: My Post Title
type: writeup          # writeup | cve | research
tags: Web, XSS, Auth
excerpt: One-sentence summary shown on the index card.
---
```

- **Publication date** is read automatically from the first git commit that added the file.
- **Images**: put them in `docs/blog/<slug>/img/` and reference them as `![alt](img/screenshot.png)`.
- **Type** controls the badge color and filter button on the index: `writeup` (green), `cve` (red), `research` (purple).
